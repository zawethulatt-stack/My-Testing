Frequently Asked Questions

ငွေပေးချေမှုစနစ်များ +
KPay, Wave Money နှင့် Mobile Banking များဖြင့် ပေးချေနိုင်ပါသည်။
ပစ္စည်းပိုဆောင်ချိန် +
ရန်ကုန်အတွင်း ၁ ရက်မှ ၂ ရက်၊ နယ်ဝေးများသို ၃ ရက်မှ ၅ ရက်အတွင်း ပိုဆောင်ပေးပါသည်။
Order ကို track လုပ်လိုရမလား? +
ရပါသည်။ သင့် Order ပိုဆောင်လိုက်သည်နှင့် Tracking Number ကို SMS ပိုပေးပါမည်။
ပစ္စည်းကို ဘယ်လိုမှာယူရမလဲ? +
Website ပေါ်ရှိ 'Add to Cart' ကိုနှိပ်ပြီး Checkout page တွင် မှာယူနိုင်သလို Facebook Messenger မှလည်း မှာယူနိုင်ပါတယ်။
ပစ္စည်းအရည်အသွေး စစ်ဆေးပြီးမှ လက်ခံရရှိနိုင်ပါသလား? +
ရန်ကုန်မြိုတွင်း ပစ္စည်းရောက်မှ ငွေချေစနစ်ဖြင့် မှာယူသူများအနေဖြင့် ပစ္စည်းကို စစ်ဆေးပြီးမှ ငွေချေနိုင်ပါသည်။
ပစ္စည်းအာမခံ (Warranty) ဘယ်လောက်ပေးပါသလဲ? +
ပစ္စည်းအမျိုးအစားပေါ် မူတည်၍ ၆ လမှ ၁ နှစ်အထိ အာမခံ ပေးထားပါသည်။
ပစ္စည်းချိုယွင်းချက်ရှိပါက လဲလှယ်ပေးပါသလား? +
ဝယ်ယူပြီး ၇ ရက်အတွင်း စက်ပိုင်းဆိုင်ရာ ချိုယွင်းချက်ရှိပါက အသစ်တစ်လုံး ချက်ချင်းလဲလှယ်ပေးပါသည်။
Installment (အရစ်ကျစနစ်) ရပါသလား? +
ဟုတ်ကဲ့၊ CB, KBZ, AYA စသည့် ဘဏ်ကတ်များဖြင့် အရစ်ကျ ဝယ်ယူနိုင်သော စနစ်များကို စီစဉ်ပေးထားပါသည်။
KPay ဖြင့်ပေးချေပါက ရာခိုင်နှုန်း (%) ထပ်ပေးရပါသလား? +
မလိုပါ။ သတ်မှတ်ထားသည့် ပစ္စည်းတန်ဖိုးအတိုင်းသာ ပေးချေရပါမည်။
Deli ခ (ပိုဆောင်ခ) ဘယ်လောက်ပေးရမလဲ? +
မြိုနယ်နှင့် အကွာအဝေးပေါ် မူတည်၍ ၃,၀၀၀ ကျပ်မှ စတင်ပါသည်။
ပစ္စည်းကို ကိုယ်တိုင်လာရောက် ဝယ်ယူလိုရပါသလား? +
ရပါသည်။ ကျွန်ုပ်တို၏ Showroom လိပ်စာအတိုင်း ရုံးချိန်အတွင်း လာရောက် ဝယ်ယူနိုင်ပါသည်။
Original ပစ္စည်းဖြစ်ကြောင်း ဘယ်လိုစစ်ဆေးရမလဲ? +
ပစ္စည်းဘူးပေါ်ရှိ QR Code သိုမဟုတ် Serial Number ကို သက်ဆိုင်ရာ Brand Website များတွင် စစ်ဆေးနိုင်ပါသည်။
Laptop ဝယ်ယူပါက Window နှင့် Software များ တင်ပေးပါသလား? +
ဟုတ်ကဲ့၊ အခြေခံလိုအပ်သော Software များနှင့် Window ကို Free တင်ပေးပါသည်။
Stock ပြတ်နေတဲ့ ပစ္စည်းတွေကို Pre-order မှာလိုရပါသလား? +
ရရှိနိုင်ပါသည်။ ပစ္စည်းအမျိုးအစားပေါ် မူတည်၍ ၂ ပတ်မှ ၄ ပတ်အတွင်း ရရှိအောင် စီစဉ်ပေးပါသည်။
ပစ္စည်းအသုံးပြုပုံ နည်းလမ်းကို ဘယ်မှာမေးလိုရမလဲ? +
ကျွန်ုပ်တို၏ Customer Service ဖုန်းနံပါတ် သိုမဟုတ် Messenger သို ၂၄ နာရီပတ်လုံး ဆက်သွယ်မေးမြန်းနိုင်ပါသည်။
